# E-commerce Product Manager

A React application for e-commerce store owners to manage products and variants with drag-and-drop functionality.

## Features

### Product List
- Display products with their variants
- Apply discounts (flat or percentage) at product or variant level
- Drag and drop to reorder products and variants
- Remove products (disabled when only one product exists)
- Show/hide variants button (only for multi-variant products)

### Product Picker
- Search products by name
- Select multiple products and variants
- Infinite scroll pagination
- Replace existing products when editing
- Prevent duplicate product selection

### Add Product
- Add empty products to the list
- Default variant with zero price
